## Evaluation

Submissions are evaluated on the **area under the ROC curve (ROC-AUC)** between the predicted probability and the observed target.

### Submission File Format

For each `id` in the test set, predict a probability for the `Heart Disease` target.

Example:

```csv
id,Heart Disease
630000,0.2
630001,0.3
630002,0.1
etc.
```

The goal of the Tabular Playground Series is to provide the Kaggle community with lightweight challenges to help develop machine learning and data science skills.
